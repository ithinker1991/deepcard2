## 🔧 技术架构

- **后端**: Python + FastAPI + SQLite + Redis
- **前端**: React + Next.js + TypeScript + Tailwind CSS
- **LLM**: 支持多厂商（OpenAI、DeepSeek、文心一言等）
- **架构**: DDD领域驱动设计

## 📝 开发原则

1. **Happy Path优先**: 先实现核心功能，后续完善边界情况
2. **渐进式开发**: 每个里程碑都有可演示的功能
3. **用户导向**: 以LLM生成为核心价值
4. **简单适用**: 避免过度设计和抽象
